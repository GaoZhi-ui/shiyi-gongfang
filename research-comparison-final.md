## 二、差异化优势（我们有，他们没有的）

| 独有功能 | 说明 | 竞品缺失 |
|:---------|:------|:---------|
| **Harness 工具调用增强** | AI 调用工具前预检/后验/重试/统计 | 任何竞品都没有 |
| **Agentic Workflow** | AI 自主 plan→draft→review→revise→save | Sudowrite 有 Story Engine 但功能不同 |
| **RAG 向量知识库** | 章节自动向量化，AI 对话时检索上下文 | 无竞品集成 |
| **伏笔追踪** | 设置/回收/状态/强度/预期章节差 | NovelWriter 有 notes 但不是专门的伏笔系统 |
| **Git 集成** | 项目自动 git init + 自动 commit + 分支管理 | 无竞品直接集成 git |
| **风格检查** | 5 条可扩展规则 | Sudowrite 有类似但封闭 |
| **插件系统** | 4 个钩子点 + 热加载 | 仅 Obsidian 有 |
| **层级标签** | Notable 风格的无限嵌套树 | NovelWriter 有标签但不是层级 |
| **灵感捕获** | localStorage + 编辑器浮动悬浮窗 | 无竞品有 |
| **写作规范 (writing-guide)** | 项目级风格/禁用词/角色名注入 AI 对话 | 无竞品有 |

## 四、同类型开源工具对比结论

| 对比项 | 写作助手工坊 | NovelWriter | Manuskript |
|:-------|:------------:|:-----------:|:----------:|
| 开源 | ✅ MIT | ✅ GPLv3 | ✅ GPLv3 |
| 代码量 | ~15,000 行 | ~50,000 行 | ~80,000 行 |
| 技术栈 | Python + JS | Python (Qt) | Python (Qt) |
| AI 集成 | ✅ 深度集成 | ❌ 无 | ❌ 无 |
| 功能密度 | 高 (60+ 功能) | 中 (30+ 功能) | 中 (25+ 功能) |
| 学习曲线 | 低 | 中 | 高 |
| 安装包大小 | 45 MB | 15 MB | 30 MB |
| 社区活跃度 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |

NovelWriter 和 Manuskript 都是纯 Qt 桌面应用，没有 AI、没有插件系统、没有 Web 界面。**写作助手工坊在功能密度上已经超过这两个成熟开源项目。**
