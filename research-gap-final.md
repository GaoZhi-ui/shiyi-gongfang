## 二、剩余差距

### ① EPUB 导出格式

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 半天 |
| **难度** | ⭐ |
| **做法** | 在 `core/export_consumers.py` 加 `EpubConsumer`，用 `ebooklib` 库 |
| **优先级** | P1（上线后可补） |

### ② 角色档案模板 / 人物卡片UI

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 1 天 |
| **难度** | ⭐⭐ |
| **做法** | 当前角色API已有name/description/traits字段，前端缺一个排版好的"人物卡片"视图。Scrivener风格：左侧字段列表（年龄/种族/职业/性格/背景），右侧自由描述区 |
| **优先级** | P1 |

### ③ 编译/导出为完整电子书（多章合并+目录+封面）

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 1-2 天 |
| **难度** | ⭐⭐ |
| **做法** | 在 `routers/export.py` 加 `POST /api/v1/export/ebook`，把多个章节合并为单个 EPUB 文件，自动生成目录+封面页 |
| **优先级** | P2 |

### ④ 命名生成器（角色/地名）

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 半天 |
| **难度** | ⭐ |
| **做法** | 调用AI生成：用户选类型（角色名/地名/组织名）+ 风格（东方/西方/奇幻），AI返回5个建议。接入现有的chat API即可 |
| **优先级** | P2 |

### ⑤ 文字转语音 / 朗读功能

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 半天 |
| **难度** | ⭐ |
| **做法** | 前端用 Web Speech API：`speechSynthesis.speak(new SpeechSynthesisUtterance(text))`，免安装，浏览器自带 |
| **优先级** | P2 |

### ⑥ 日历视图 / 写作计划

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 1 天 |
| **难度** | ⭐⭐ |
| **做法** | 融合现有的目标追踪(goals)和统计(stats)，在右侧面板加一个简易月历视图，标出每天写了多少字 |
| **优先级** | P2 |

### ⑦ 导出为图片（长图分享）

| 项目 | 说明 |
|:-----|:------|
| **工作量** | 半天 |
| **难度** | ⭐ |
| **做法** | 前端用 `html2canvas` 库，把选中文本渲染为带格式的长图，一键下载。适合分享到社交平台 |
| **优先级** | P3 |

---

## 三、总结

| 等级 | 项数 | 总工作量 |
|:-----|:----:|:---------|
| ⭐ 低（半天） | 4 | 2 天 |
| ⭐⭐ 中（1天） | 3 | 3 天 |
| **合计** | **7 项** | **约 5 天** |

最推荐的三项：**EPUB导出 → 角色卡片UI → 日历写作计划**
