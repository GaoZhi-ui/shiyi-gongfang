## ⚠️ 关键发现：主页面 IIFE 因语法错误未执行

**严重性：P0 — 阻塞级**

主页面 `index.html` 中最大的 `<script>` 块（`(function(){...})()`）因 **括号不平衡（多出 2 个 `(`）** 导致整个 IIFE 未执行。

**影响范围：**
- CodeMirror 编辑器未自动初始化
- 所有快捷键绑定失效（Ctrl+S、Ctrl+N、Ctrl+↑/↓）
- TTS 朗读、导出图片功能的事件绑定未注册
- 侧边栏点击无响应（handleToolClick、setPanelContent 未定义）
- 角色卡片、日历等右侧面板内容无法通过点击加载
- 项目加载、API 连接检查等自动流程未触发

**根因分析：** IIFE 结尾附近存在未匹配的 `(`。IIFE 闭合 `})();` 时 JavaScript 解析器检测到还有未闭合的 `(`，抛出 `Unexpected token ')'`。

**建议修复：** 使用 ESLint 或 `node --check` 检查 `index.html` 中内联脚本的语法。检查在模板字符串或字符串拼接中是否有多余的 `(`。
